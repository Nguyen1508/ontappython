ÔN TẬP PYTHON

Ngôn ngữ máy thường ở dạng nhị phân hay hợp ngữ → Chương trình khó viết và khó hiểu.

Ngôn ngữ lập trình bậc cao với câu lệnh gần với ngôn ngữ tự nhiên → Cần chương trình dịch để máy tính hiểu

Một số ngôn ngữ lập trình: Java, C/C++, Python, ...

Biến là tên của một vùng nhớ dùng để lưu trữ giá trị (dữ liệu) và giá trị đó có thể thay đổi.

Cú pháp

<biến> = <giá trị>

Ví dụ:

x=5
Chức năng Mã lệnh
Nhập dữ liệu 
<biến> = input(<Dòng thông báo>)
Xuất dữ liệu 
print()
Tìm kiểu dữ liệu của một biến 
type()
Chuyển đổi sang số nguyên 
int()
Chuyển đổi sang số thực 
float()
Chuyển đổi sang xâu ký tự 
str()
Ví dụ 
x = input("Nhập số lượng") 
print("Kiểu dữ liệu của x: ",type(x)) 
print("Kiểu dữ liệu của x: ",type(int(x)))

Cú pháp:

if <điều kiện>: 
     <khối lệnh 1> 
else: 
     <khối lệnh 2>

Ví dụ 1:

if k<= 0: 
      print("Bạn nhập sai rồi!")

Ví dụ 2: Viết chương trình nhập vào một số. Kiểm tra và in ra màn hình đó là số chẵn hay lẻ.

Ví dụ 3: Viết chương trình nhập vào 3 số a, b, c. Kiểm tra 3 số đó có phải là 3 cạnh của 1 tam giác hay không?

Cú pháp:

for <i> in range(n) 
     <khối lệnh>

Ví dụ 1:

for k in range(10): 
      s=s+k

Ví dụ 2: Viết chương trình nhập từ bàn phím số tự nhiên n và in ra kết quả S= 1 + 2 + .... + n

Ví dụ 3: Viết chương trình nhập từ bàn phím số tự nhiên n và in ra kết quả S= 1^3 + 2^3 + .... + n^3
